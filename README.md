# 🐦 PyTweet - Twitter Clone

A lightweight Twitter-like social media platform built with **FastAPI (Backend)** and **Vanilla JavaScript + TailwindCSS (Frontend)**.

---

## 🚀 Tech Stack
- **Backend:** Python 3.12+, FastAPI, SQLModel (SQLAlchemy)
- **Database:** PostgreSQL (or SQLite for local development)
- **Frontend:** HTML5, Vanilla JavaScript, TailwindCSS (CDN)
- **Authentication:** BCrypt password hashing
